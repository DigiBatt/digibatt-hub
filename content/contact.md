## Get in Touch

We welcome questions, feedback and contributions from the battery modelling community.

### Submitting a Resource

The quickest way to add a resource to the hub is via a GitHub Issue. Use the **Submit** link in the navigation bar, or go directly to the [submissions page](https://github.com/FrancescaWatson/digibatt-hub-hugo/issues/new/choose) and choose the *Submit a Record* template.

### Reporting Issues

Found a broken link or incorrect information? Please [open an issue](https://github.com/FrancescaWatson/digibatt-hub-hugo/issues) on GitHub and we will take a look.

### Project Website

For general enquiries about the DigiBatt project, please visit [digibatt.org](https://digibatt.org).
