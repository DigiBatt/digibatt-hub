This record contains battery datasets exported from different cyclers. It is intended to be used for testing and validating battery data pipelines and workflow automation. The datasets cover various battery chemistries and test types. Some contain common bugs that can occur during testing (e.g. bumping the connection cables) or from software export routines. Including data with bugs is done intentionally to verify data processing robustness.
